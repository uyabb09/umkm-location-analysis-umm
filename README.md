This repository contains a **Web-based Geographic Information System (WebGIS)** project for mapping and analyzing **UMKM (Small and Medium Enterprises)** locations around **Universitas Muhammadiyah Malang (UMM)** using **real field data**.

The project aims to visualize UMKM distribution and support simple location-based analysis to identify potential business areas around the UMM region.

The system displays UMKM locations on an interactive map using latitude and longitude data collected from direct observation. Each data point represents a real UMKM location around the UMM area.
